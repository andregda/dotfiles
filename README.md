## ZSH

<p align="center">
Fast umbloated ZSH config achieved by loading compinit once a day, not every single instance.<b> No bloated plugin necessary</b>
</p>

#### Try it out with

```bash
	for i in $(seq 1 10); do time zsh -i -c exit; done
```
